# Student Result Management System (Using PHP & MySQL)

**Student Result Management System** is a web-based application built with PHP and MySQL to help educational institutions manage academic records efficiently. The system allows administrators to add, update, and publish student results securely, while students can search and view their own marks easily through a user-friendly interface. This reduces manual work, eliminates errors, and improves result accessibility.

Follow these steps to manually import the database schema using the provided SQL file:
1.Start Apache and MySQL from the XAMPP / WAMP Control Panel.

2.Open your web browser and go to:
 http://localhost/phpmyadmin

3. In phpMyAdmin, click on New from the left sidebar.
  Enter the database name: [ srms ] and click Create.

4. Select the newly created srms database.
5. Click on the Import tab from the top menu.
6. Click Choose File and select the SQL file provided with the project:
   srms.sql

7.Keep the default import settings unchanged.
8.Click Go to start the import process.
